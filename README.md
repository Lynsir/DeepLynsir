# Deep Learning with Lynsir
From: https://github.com/Lynsir/DeepLynsir
## 文件夹说明：
- **app**：用于启动应用程序的文件
- **data**：示例数据文件
- **datasets**：数据集载入处理文件
- **model**：自构模型库
- **paper**：标准模型库（论文实现）
- **util**：工具库
- **run_app.py**：启动应用程序的脚本
## 运行应用程序
```bash
python -m app.example_app --epochs=1 --batch-size=8 test
```
